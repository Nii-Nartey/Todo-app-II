# ToDo App

A simple ToDo app that helps you keep track of your tasks and manage your day effectively. This app allows you to create, update, and delete tasks, ensuring you stay organized and never miss an important task.

### Features

Create tasks with a title, description, and due date.
Mark tasks as complete or incomplete.
Edit task details such as title, description, and due date.
Delete tasks that are no longer needed.
View all your tasks in a clean and intuitive interface.

### Installation

1. Clone the repository to your local machine using the following command:

git clone https://github.com/Nii-Nartey/Todo-app-II.git

2. Navigate to the project directory: cd Todo-app-II

3. Open the index.html file in your web browser.

### How to Use

1. Add a Task: Click on the "Add Task" button and fill in the required details - title, description, and due date. Then click "Save" to add the task to your ToDo list.

2. Mark as Complete/Incomplete: When you have completed a task, click on the checkbox next to the task to mark it as complete. If you want to mark it as incomplete, simply click the checkbox again.

3. Delete a Task: When you no longer need a task, click on the "Delete" button represented by a trash icon. Confirm the action, and the task will be removed from your ToDo list.
